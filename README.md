# CRMRB

Crystal with embedded MRuby C library

## Install/Usage

```sh
$ git clone https://github.com/RyanScottLewis/crmrb.git
$ cd crmrb
$ rake build
$ ./build/exe/crmrb ./examples/hello.rb
Hello from MRuby in Crystal
```
